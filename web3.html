HyperText Markup Language je název značkovacího jazyka, který se používá pro tvorbu webových stránek. Jedná se o hlavní jazyk pro tvorbu www stránek.

Jazyk je aplikací dříve vyvinutého univerzálního značkovacího jazyka SGML. HTML se vyvíjel v závislosti na webových prohlížečích.

Jazyk je charakterizován množinou značek (tagů). Mezi značky se uzavírají části textu dokumentu a tím se určuje význam obsaženého textu. Názvy jednotlivých značek a jejich vlastní se uzavírají mezi závorky
< >. Část dokumentu, která je tvořena otevírající značkou, nějakým obsahem a ukončující značkou se nazývá element dokumentu.

Pro příklad: <b>Ahoj!</b> tvoří tučný text.

Značky jsou obvykle párové, kde je počáteční značka shodná s koncovou, jen je na konci před značkou lomítko.

Pro příklad: vytvoření odstavce textu: <p>Zde je jeden odstavec</p>.

Některé značky jsou nepárové.

Pro příklad:
  • Vytvoření vodorovné čáry: <hr>
  • Nový řádek: <br>

Značky mohou obsahovat vlastnosti, které o nich uvádí důležité informace. Koncové značky už žádnou vlastnost nemají. Pro příklad značka <a>, kde uvedeme vlastnost href říká, kam se uživatel po kliknutí na něj dostane.

Příklad: <a href=“https://google.com“>Přesměrování na google</a>

Ke značkám dále můžeme doplnit více vlastností. Příklad: otevření webové stránky v novém okně: <a href=“https://google.com“ target=“_blank“>Přesměrování na google</a>

K odkazu dále můžeme doplnit titulek, který se zobrazí po najetí kurzorem na hypertextový odkaz nebo přiřadit třídu, které můžeme definovat vlastnosti pomocí kaskádových stylů (CSS).

HTML dokument dále může obsahovat další prvky:
  • DTD direktivy: začínají pomocí <!, slouží pro zpracovatele dokumentu, v našem případě prohlížeč.
  • Komentáře: slouží jako pomocné texty pro programátora, nejedná se o obsah dokumentu, nezobrazují se a prohlížeč je ignoruje.
  • Kód skriptovacích jazyků: PHP nebo Javascript
  • Kaskádové styly
Struktura dokumentu:
  • Deklarace typu dokumentu: značka <!DOCTYPE html> která říká prohlížeči, že se jedná o HTML dokument
  • Kořenový element: značka <html> </html>, která reprezentuje celý dokument
  • Hlavička dokumentu: značka <head> </head>, obsahuje metadata (= data o jiných datech), která se vztahují k celému dokumentu. Definuje kódování, název dokumentu, autora, popis, klíčová slova, titulek dokumentu nebo kaskádové styly.
  • Tělo dokumentu: značka <body> </body>, zahrnuje vlastní obsah dokumentu.
Druhy značek:
  • Strukturální: rozvrhují strukturu dokumentu, například odstavce <p>, nadpisy <h1>, <h2>. Strukturální značky dodávají dokumentu formu.
  • Popisné: popisují povahu prvku, například nadpis <title> nebo adresa <address>.
  • Stylistické značky: určují vzhled prvku při zobrazení. Typickým příkladem je <b> tučné písmo. Od těchto značek se postupně odstupuje a používají se kaskádové styly, které vzhled popisují odděleně od obsahu dokumentu. Důvodem, proč nepoužívat tyto značky, ale kaskádové styly, může být, že jsou orientovány na prohlížení na obrazovce počítače, nepočítá se příliš pro používání dokumentu jiným způsobem, např. prohlížení pro postižené (čtečky pro slepce), mobilní zařízení atd. Kaskádové styly však umožňují definovat rozdílná zobrazení pro různá zařízení.
Základní struktura HTML dokumentu
Každý HTML soubor by měl obsahovat několik základních tagů označujících hlavičku a tělo. Těmito tagy je zaručeno, že všichni klienti (především prohlížeče) pochopí, o co v dokumentu vlastně jde. Mezi tyto základní tagy patří:

Základní struktura html
<html> začátek HTML dokumentu je párový začíná a končí dokument, jedná se však o nepovinnou značku
<head> hlavička je párový hlavička dokumentu, obsahuje další tagy určující například klíčová slova
<body> tělo stránky je párový obsahuje veškerý zobrazovaný obsah stránky
!doctype specifikace není párový udává informace o verzi html, přepíná zobrazovací módy CSS aj.

Formátování textu
V dnešních dnech se pro formátování textu na webové stránce používá spíše kaskádových stylů, které dokáží upravit formát celého dokumentu. Mezi nejpoužívanější HTML tagy upravující text patří:

Formátování textu
<b> tučné písmo párový
<i> kurziva párový
<span> úsek textu párový
<sub>, <sup> dolní/horní index párový

Bloky
Bloky slouží k oddělování jednotlivých částí stránky (nejčastěji textu)

Bloky
<p> odstavec nepovinně párový
<br> zalomení řádku nepárový
<b> oddíl párový
<h1>, <h2>,.... nadpis 1. , 2. ,...x. úrovně párový

Seznamy
Někdy též označovány jako odrážky či číslování.

Seznamy
<li> položka seznamu nepovinně párový
<ol> číslovaný seznam párový
<ul> odrážkový seznam párový

Mezi další tagy patří tagy označující odkaz (&lt; a &gt;), tagy pro vkládání různých objektů - od obrázků až po Java applety, tagy upravující formuláře apod.

Atributy
Každý tag může mít různé atributy (vlastnosti), které jej odlišují od ostatních prvků stejné kategorie. Pomáhají tak například ve formátování pomocí CSS. Mezi používané patří class, id a name. Pro přímý zápis stylu se používá vlastnost style.
